---

##  Introduction

Phishing is one of the most common cyber attacks used to steal:

- Login credentials  
- OTPs  
- Personal and sensitive information  

Attackers usually target **people, not systems**.

This report analyzes a phishing email and provides **simple awareness guidelines** that any employee can easily understand and follow.

---

##  Objective of This Report

The main goals of this report are to:

- Analyze phishing email samples  
- Identify common phishing indicators (**red flags**)  
- Classify email risk clearly  
- Explain how phishing attacks work (simple explanation)  
- Provide clear **Do’s and Don’ts** for employees  

---

-  Email Header Analyzer

-  Domain inspection  
-  Documentation using Markdown (GitHub-ready)

---


## Conclusion

Phishing attacks succeed mainly due to **lack of awareness**.

By learning how to identify phishing indicators and following simple safety practices, employees can:

- Prevent credential theft  
- Avoid account compromise  
- Reduce major security incidents
